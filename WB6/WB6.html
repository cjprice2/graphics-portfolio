Graphics Projects

Primitive Snowman
A 3D snowman made from primitives.
  • Built with Three.js and the CS559 Framework
  • Snowman assembled from spheres, cylinders, and a cone for the nose.
  • Ground plane, ambient light, and directional lighting.
  • Animated bounce and circular motion.
  • Use left mouse to rotate the camera, use right mouse to move the camera, and use the scroll wheel to zoom in and out.
Object Museum
A museum of 3D objects.
  • Built with Three.js and the CS559 Framework
  • SpongeBob object assembled from spheres, boxes, and cylinders.
  • Other objects are imported from OBJ files.
  • Camera controls: left-mouse drag = orbit, right-mouse drag = pan, scroll = zoom.
  • Use the buttons to switch between cameras.
