Quadcopter Game
A quadcopter arcade game where you pilot a blue drone and rack up points by shooting down enemies.
  • Move with WASD keys.
  • Aim with the mouse pointer and left-click to shoot.
  • Enemy quadcopters are decorative and cannot harm you.
  • Earn 1 point for every enemy you destroy.
  • You can play as long as you want, the game never ends.
  • Built with the HTML5 Canvas 2D API.
